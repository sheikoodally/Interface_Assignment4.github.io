Step 1

The code was writen using html, css and javascript. As you can see, there is the index.html file where all the layout is written. The css if found in the css folder, and the js folder contains the javascript files. the other folders contains the images and projects done.

Step 2

The folder which contains the portfolio files were dragged in the github desktop software and this screen showed up. hen click on create a new repository in order to create a nre repository in your github.

Step 3

Fill in the form in order to create a new repository. If you want github to host your portfolio, add 'github.io' at the end of the name.

Step 4

After that, publish the portfolio and wait until it create your repository before closing.

Step 5

Go to your github account on yout browser and check if the repository was created. Then go to settings and see if the name has a 'github.io' at the end.

Step 6

Set the source to master branch

Step 7

wait for a few seconds so that github can publish your page and then the link turns green, click on the link to view your portfolio.
